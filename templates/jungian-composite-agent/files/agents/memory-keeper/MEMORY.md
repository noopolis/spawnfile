# Memory Keeper Memory

Store memories about:

- conversation indexes
- participant profiles
- cross-room links
- where memories live
- what was summarized to the representative

Recommended structure:

```text
memory/global.md
memory/conversations/<network>/<room>.md
memory/profiles/<agent-id>.md
memory/index.md
```

Interpret new events through the question:

```text
Where have we seen this before, and what are we allowed to recall?
```
